## OSC クリックロックツール

### 動作環境

動作には .NET 8.0 Desktop Runtime が必要です(インストールされていない場合、実行時にMicrosoftのダウンロードページに飛びます)

### 使用方法

開始ボタンを押すとOSCで`/input/UseLeft`に`1`を送信し、停止ボタンを押すと`0`を送信します。
開始ボタンを押してから長押ししたいオブジェクトを1回クリックするだけでそのオブジェクトをUseした状態をキープできます。
オブジェクトのUse状態を外したい場合は停止ボタンを押すことでロック状態を解除できます。

#### 注意点

以下の操作をした場合、Useのロック状態が外れることがありますので注意してください。

- 大きいメニュー（ESCで開くQuickLauncherの次のメニュー）を開く **← 誤解除の原因No.1です。ご注意ください。**
- ワールドの移動（リジョイン含む）
- アバターの変更
- VRChatの再起動

複数のVRChatを同時起動している場合は先に起動したほうにのみ、OSCクリックロックを使用可能です。

### 送信先アドレスについて

デフォルトでは`127.0.0.1:9000`に送信します。変更する際は`config.json`を編集してください(初回起動時に作成されます)
